# Gruvbox GNOME Dotfiles

Gruvbox GTK theme setup for GNOME with extensions, backgrounds, and terminal configs.

## Theme Settings
- Color scheme: Hard (default)
- Style: Borderless windows, macOS buttons, floating panel, 2px outline (shell/dash)

## Extensions
- openbar
- vertical-workspaces
- dash2dock-lite
- appindicatorsupport
- clipboard-history
- gTile
- monitor (Astra Monitor)
- gsconnect

## Install

```sh
git clone https://github.com/idunkoncode/Gnome-dotfiles.git
cd Gnome-dotfiles
chmod +x install.sh
./install.sh
```

## Structure

```
config/         GTK 3/4 configuration
extensions/     GNOME Shell extensions
themes/         Gruvbox-Dark GTK theme
backgrounds/    Wallpapers
fonts/          Fonts
dconf/          GNOME settings dump
```
